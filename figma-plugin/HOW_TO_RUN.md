# PAT 2.0 Login Screen — Figma Plugin

## How to run (takes ~30 seconds)

1. Open **Figma desktop app** (or figma.com in browser)
2. Create a new **blank file** (or open an existing one)
3. Go to menu: **Main menu → Plugins → Development → New Plugin...**
4. Click **"Click here to choose a manifest.json file"**
5. Select the `manifest.json` in this folder (`figma-plugin/`)
6. Back in Figma: **Plugins → Development → PAT 2.0 Login Screen → Run**
7. The full login screen frame is built on your canvas automatically ✅

## What gets created

A `1440 × 900` frame named **"PAT 2.0 – Login"** with:

| Left panel (816px) | Right panel (624px) |
|---|---|
| Dark navy (#192040) background | Clean white / off-white card |
| Dot-grid texture | "Sign in" heading |
| NeoPAT circular logo | Email Address input field |
| NeoPAT + Placement Administration title | Continue button (navy) |
| Feature cards 2×2 grid | Student Access section |
| Drive Management, Student Management, | App Store + Google Play badges |
| Communication, Analytics & Reports | Copyright footer |
| iamneo / NIIT brand footer | |
